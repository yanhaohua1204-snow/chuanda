# 使用指南

## 快速开始

这是一个完全实现的移动端穿搭助手 Demo，所有功能都已经可以使用。

## 交互流程

### 1. 抖音挂载页（启动页）
- **进入方式**：应用启动时默认显示
- **交互**：点击屏幕任意位置进入下一步
- **效果**：模拟从抖音视频页面的挂载入口进入应用

### 2. AI 思考页
- **自动触发**：从挂载页进入后自动显示
- **显示内容**："思考中..." 文字提示
- **持续时间**：2 秒后自动跳转到 AI 回答页

### 3. AI 回答页（主页）
- **AI 打字机效果**：
  - 自动逐字显示分析结果
  - 速度：每个字符间隔 80ms
  - 内容：穿搭风格分析和搭配建议

- **穿搭视频分析卡片**：
  - 显示视频标题和风格标签
  - 实时打字显示分析文本
  
- **单品识别衣橱匹配卡片**：
  - 5 个单品格子展示匹配结果
  - 绿色标签：来自"我的衣橱"
  - 粉色标签：DaDou 推荐单品
  - 换一换按钮：点击可更换推荐（视觉反馈）

- **底部对话框**：
  - 白色输入框（可点击）
  - 绿色发送按钮

- **底部导航栏**：
  - AI搭配（当前页，绿色高亮）
  - 我的衣橱（点击切换）
  - 穿搭手帐（点击切换）

### 4. 我的衣橱页
- **内容区域**：
  - 显示所有衣物单品
  - 按分类展示（上衣、裤子、包包、鞋子等）
  - 支持垂直滚动浏览更多单品

- **底部导航**：
  - "我的衣橱"高亮显示（绿色）
  - 可切换到其他页面

### 5. 穿搭手帐页
- **时间线展示**：
  - "今天"、"05.20"、"05.16" 等日期标签
  - 每个日期下展示当天的穿搭记录

- **穿搭卡片**：
  - 左侧：穿搭照片
  - 右侧：单品明细列表
  - 支持垂直滚动查看历史记录

- **底部导航**：
  - "穿搭手帐"高亮显示（绿色）
  - 可切换到其他页面

- **新增按钮**：
  - 右下角绿色圆形按钮
  - 点击可添加新的穿搭记录（视觉反馈）

## 页面滚动

所有列表内容区域都支持滚动：

## 开发说明

### 文件结构
```
src/app/
├── App.tsx                 # 主应用，管理路由和页面切换
└── components/
    ├── DouyinEntry.tsx     # 抖音挂载页
    ├── AIThinking.tsx      # AI 思考页
    ├── AIAnswer.tsx        # AI 回答页（含打字机效果）
    ├── Wardrobe.tsx        # 我的衣橱页

```

### 状态管理
- 使用简单的 React state 管理当前页面
- 页面切换通过回调函数实现
- 打字机效果使用 useEffect 和 setInterval

### 添加新功能
1. 在对应组件中添加交互逻辑
2. 使用 onClick 等事件处理器
3. 更新 state 触发 UI 变化

## 常见问题

**Q: 为什么看不到打字机效果？**  
A: 打字机效果在 AI 回答页自动播放，请确保已经从抖音挂载页点击进入并等待 AI 思考页完成。

**Q: 底部导航点击没反应？**  
A: 底部导航已经完全实现，点击会切换页面。请确保点击的是导航按钮区域。

**Q: 页面滚动不流畅？**  
A: 请确保使用移动设备或浏览器的移动模拟模式，桌面鼠标滚轮可能体验不佳。

**Q: 如何重新开始体验？**  
A: 刷新浏览器页面即可从抖音挂载页重新开始。
